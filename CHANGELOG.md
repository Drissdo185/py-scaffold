## [0.1.0] - 2026-01-03

### Added
- Initial release of py-scaffold
- Backend API template with layered architecture
- Model, Repository, Service, Controller, and DTO layers
- YAML configuration support with Pydantic
- Interactive template selection
- CLI interface with Click
- Project generation with customizable output directory
- Force overwrite option for existing directories
- Type hints and mypy support
- Black and isort configuration
- Pytest setup for testing

### Features
- Production-ready project templates
- Clean separation of concerns
- Type-safe configuration management
- Industry-standard project structure
